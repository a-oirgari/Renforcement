# Semaine 1 - JavaScript : Tableaux et Objets

## Objectifs

- Maitriser les methodes fonctionnelles des tableaux (map, filter, reduce, sort, find)
- Manipuler des objets imbriques et des tableaux d'objets
- Implementer une logique metier realiste avec uniquement JavaScript natif
- Ecrire du code lisible, bien structure, sans boucles imperatives inutiles

## Organisation

| Jour      | Theme                                  |
|-----------|----------------------------------------|
| Jour 01   | Sauvetage : bases JS (avec solutions)  |
| Jour 02   | Methodes de tableaux et pipelines      |
| Jour 03   | Objets complexes et transformations    |
| Jour 04   | CRUD, logique metier, agregations      |
| Jour 05   | Mini-projet : Gestionnaire de produits |

## Rappels importants

- Aucune librairie externe : JavaScript pur uniquement
- Privilegier les methodes fonctionnelles aux boucles for classiques
- Chaque fonction doit avoir une responsabilite unique
- Tester vos fonctions avec des `console.log` ou `console.assert`
