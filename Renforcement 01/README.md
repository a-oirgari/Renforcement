# DWWM - Programme de Renforcement Pedagogique

Formation Developpeur Web et Web Mobile | 7 semaines | Referentiel TP-01280

## Structure du depot

```
semaine-01-javascript-tableaux-objets/
semaine-02-uml-sql/
semaine-03-php-poo-bases/
semaine-04-php-poo-avance-tsql/
semaine-05-javascript-async/
semaine-06-fullstack/
semaine-07-revision-soutenance/
```

Chaque semaine contient :
- `Jour 01-sauvetage/` : exercices simplifies avec solutions (apprenants en difficulte)
- `Jour 02/`, `Jour 03/`, `Jour 04/` : exercices de renforcement (2-3 par jour)
- `Jour 05-mini-projet/` : enonce du mini-projet hebdomadaire

## Niveaux de difficulte

- Jour 01 sauvetage : accessible, guide, avec corrections
- Jour 02 a Jour 04 : intermediaire a avance, autonomie attendue
- Jour 05 : synthese, notation sur 20

## Competences DWWM couvertes

| Semaine | Competences |
|---------|-------------|
| 1       | C3 - Interface utilisateur dynamique (JS) |
| 2       | C1, C5 - Maquettage, Base de donnees |
| 3       | C5, C6, C7 - BDD, Acces donnees, Backend PHP |
| 4       | C6, C7 - Backend PHP avance, T-SQL |
| 5       | C3 - JavaScript asynchrone, Fetch API |
| 6       | C1 a C7 - Full-stack integration |
| 7       | Toutes - Revision et preparation soutenance |
