## Aide a la redaction du dossier projet (1h)

Structure minimale attendue pour le dossier projet de la soutenance finale :

1. Presentation du projet
   - Contexte et objectifs
   - Technologies utilisees et justification des choix

2. Conception
   - Diagramme de cas d utilisation (capture d ecran)
   - Diagramme de classes (capture d ecran)
   - Schema de la base de donnees

3. Architecture technique
   - Structure du projet (arborescence commentee)
   - Description du pattern architectural utilise (MVC, etc.)

4. Fonctionnalites implementees
   - Liste avec captures d ecran ou GIF animes
   - Pour chaque fonctionnalite : description, code source pertinent, explication

5. Securite
   - Mesures implementees (XSS, CSRF, SQLi, hashage des mots de passe)
   - Exemple de code pour chaque mesure

6. Difficultes rencontrees et solutions
   - Au moins 2 problemes techniques et comment ils ont ete resolus

7. Bilan et perspectives
   - Ce qui a ete fait vs ce qui etait prevu
   - Ce qui serait fait differemment avec plus de temps
