# Jour 03 — Entrainement oral et preparation du fil rouge

## Objectif

S exercer a presenter son projet fil rouge et recevoir des retours constructifs.

## Partie 1 (45min) : Quiz JavaScript et full-stack

Reprendre les questions 33 a 40 du quiz du Jour 02 plus les questions specifiques au projet :

- Comment avez-vous organise votre code JavaScript ?
- Montrer un exemple de `fetch` dans votre projet et l expliquer ligne par ligne.
- Comment avez-vous gere les erreurs reseau dans votre frontend ?
- Qu est-ce que le debounce et l avez-vous utilise ?

## Partie 2 (45min) : Presentation du fil rouge (par binomes)

Chaque apprenant presente son projet fil rouge a son binome.
Le binome joue le role du jury et pose des questions.

### Trame de presentation (10 minutes maximum)

1. Contexte et objectif du projet (1 min)
   - Quel est le probleme resolu ?
   - Qui sont les utilisateurs ?

2. Demonstration technique (5 min)
   - Parcourir les fonctionnalites principales
   - Montrer une fonctionnalite qui vous rend fier(e)
   - Montrer la gestion d erreurs

3. Architecture et choix techniques (2 min)
   - Pourquoi ces technologies ?
   - Comment avez-vous structure le code ?

4. Difficultes et apprentissages (2 min)
   - Un probleme difficile rencontre et comment il a ete resolu
   - Ce que vous feriez differemment

### Feedback binome (5 minutes)

- Ce qui etait clair et bien explique
- Ce qui manquait ou etait confus
- Une question que le jury pourrait poser et qui n a pas ete abordee

## Partie 3 (30min) : Travail sur le dossier projet

Sur la base des retours recus, identifier et combler les lacunes du dossier :
- Les captures d ecran manquantes
- Les sections pas assez detaillees
- Les justifications de choix a renforcer
