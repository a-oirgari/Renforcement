# Semaine 3 - PHP : Programmation Orientee Objet (Bases)

## Objectifs

- Definir des classes avec proprietes privees et methodes publiques
- Implementer correctement l'encapsulation (getters/setters avec validation)
- Utiliser l'heritage et les classes abstraites
- Structurer des mini-applications en POO

## Conventions a respecter

- Proprietes privees (`private`) par defaut, protected si heritage prevu
- Constructeur initialise toutes les proprietes obligatoires
- Methodes de validation separees quand la logique est complexe
- Un fichier PHP par classe
- Namespace : `App\Models`

## Organisation

| Jour     | Theme                                         |
|----------|-----------------------------------------------|
| Jour 01    | Sauvetage : PHP syntaxe + premiere classe     |
| Jour 02    | Encapsulation et classes metier               |
| Jour 03 | Heritage, classes abstraites et polymorphisme |
| Jour 04    | SQL avance (jointures, sous-requetes)         |
| Jour 05 | Mini-projet : Systeme scolaire                |
