# Semaine 6 — Full-Stack : PHP + JavaScript + UML

## Format special

5 jours de 2 heures chacun (10 heures au total).
Cette semaine integre toutes les competences acquises dans un projet full-stack complet.

| Jour | Contenu |
|------|---------|
| Jour 01 | Conception UML + architecture MVC + Git |
| Jour 02 | Backend PHP : Models PDO, Controllers, auth, securite |
| Jour 03 | Frontend JS : DOM, Fetch/AJAX, panier, LocalStorage |
| Jour 04 | Integration, tests, securite (XSS/CSRF/SQLi), Bootstrap |
| Jour 05 | Soutenance individuelle 10-15 minutes |

## Jour 01 — Programme de sauvetage (2h)

Revision globale PHP + SQL + JS, schema client-serveur, TP guide formulaire->PHP->BDD.
Voir `Jour 01-sauvetage/` pour le contenu detaille.
